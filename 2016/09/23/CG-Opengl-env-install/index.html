ubuntu下opengl的开发环境安装

codeblocks

  • 安装
    • 软件安装: sudo apt-get install codeblocks

  • 运行说明

    • 运行主要是选择console application然后设置链接的库就可以了

  • 运行glut

    • 在上面已经安装了
    • 查找软件安装位置 dpkg -L freeglut3,得到我的安装目录位置在/usr/lib/x86_64-linux-gnu

    • 复制到/usr/lib目录下（要根据得到的目录的地址进行变换）

      • sudo cp /usr/lib/x86_64-linux-gnu/libGL.so /usr/lib/libGL.so
      • sudo cp /usr/lib/x86_64-linux-gnu/libGLU.so /usr/lib/libGLU.so
      • sudo cp /usr/lib/x86_64-linux-gnu/libglut.so /usr/lib/libglut.so

    • 运行

      • 打开codeblocks-> 新建project -> console application
      • [settings] -> Compiler 选择 GNU GCC Compiler
      • [settings] -> Compiler 选择 Liker settings -> Link library (填写如下的地址)
        • /usr/lib/libglut.so
        • /usr/lib/libGLU.so
        • /usr/lib/libGL.so

    • 测试代码可以用上面用的代码

  • 运行glew与glfw

    • glew 安装

      • apt-cache search glew //查找依赖
      • sudo apt-get install 。。。(…里面填写在apt-cache里面显示的软件)

    • glfw3 安装

      • apt-cache search glew
      • sudo apt-get install libglfw3-dev libglfw3//只安装glfw3

    • 复制

      • glew: 在上面安装完后应该在/usr/include/GL下面有一个glew.h的文，同时在通过dpkg -L libglew1.10（我的是libglew1.10）查看相关的so文件在哪里，我的是在/usr/lib/x86_64-linux-gnu/libGLEW.so.1.10;
      • glfw3: 操作与glew一样，先查看安装在哪里，
      • 复制 sudo cp /usr/lib/x86_64-linux-gnu/libGLEW.so.10 /usr/lib sudo cp /usr/lib/x86_64-linux-gnu/libglfw.so.3

    • 设置

      • 运行同运行glut的一样，但是在设置链接时也就是[settings->linker settings]这里，要多设置上/usr/lib/libglfw.so.3 /usr/lib/libGLEW.so.1.10
